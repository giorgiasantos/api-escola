# api-escola
Exercicio de API REST com DTOS

## Funcionalidades:

1. Cadastro de alunos.
2. Cadastro de curso.
3. Cadastro de professor.
4. Listar todos os alunos.
5. Listar todos os professores.
6. Listar todos os cursos.
7. Atualizar curso do aluno.
8. Deletar curso.
9. Deletar aluno.
10. Deletar professor.
